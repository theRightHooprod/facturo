# Facturo

Open source Mexican's SAT invoices manager. It Loads an entire folder and list's invoices and its metadata.

Requirements

`nodejs > v22.15.0`

To compile from source just:

`npm i`

`npm run dev`

It uses electron to be compatible with Window 11 desktop app.
